HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan Tag. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web. HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar.

  1. Tag adalah sebauh penanda awalan dan akhiran dari sebuah elemen di HTML. Tag dibuat dengan kurung siku (<...>), lalu di dalamnya berisi nama tag dan kadang juga ditambahkan dengan atribut.

  2. Elemen dalam HTML adalah sebuah komponen yang menyusun dokumen HTML. Elemen kadang juga disebut sebagai node, karena ia merupakan salah satu jenis node yang menyusun dokumen HTML dalam diagram HTML tree.

  3. Atribut adalah kata khusus yang berada di dalam tag pembuka. Atribut juga disebut sebagai modifier yang akan menentukan perliaku dari elemen.

Pada materi ini, kita belajar beberapa tag:

  1. <p> untuk membuat paragraf
  2. <hr> untuk membuat garis lurus
  3. <br> untuk membuat baris baru
  4. <div> untuk membuat paragraf di luar artikel atau layaout
  5. <pre> untuk membuat paragraf dengan format yang sudah ditentukan.
Ini adalah tag-tag dasar yang perlu diingat dalam membuat paragraf di HTML.

Form di HTML dapat kita buat dengan tag.

Tag ini memiliki beberapa atribut yang harus diberikan, seperti:

  • Action untuk menentukan hasil yang akan dilakukan saat data dikirim

  • Metode pengiriman data.

Contoh:

<form> action="prosess.php" method="GET">

<!-- form field di sini -->

</form>

Di HTML terdapat beberapa elemen yang khusus untuk melakukan ini. Yakni elemen semantik seperti <header>,<aside>,<footer>,<article>, dll.

Elemen semantik sebenarnya tidak hanya digunakan untuk membuat layout saja. Ia juga digunakan untuk membuat elemen lain.

Di awal-awal hadirnya HTML dulu, elemen semantik belum ada. Orang-orang membuat layout dengan menggunakan tag yang salah. Ada yang membuatnya dengan tag <tabel> dan ada juga dengan tag <div>.

Ini sebenarnya tidak sepenuhnya salah, karena membuat layout dengan kedua tag itu benar-benar bisa. Tapi, ini bukanlah cara yang baik dan akan membuat kode HTML kita sulit terbaca. Karena itu, hadirlah elemen semantik sebagai solusi.

CSS adalah bahasa kedua setelah HTML yang harus dipelajari seorang web developer.

Pengetahuan tentang CSS sangatlah penting bagi seorang web developer, karena dengan CSS kita bisa:

  1. Halaman landing page yang menarik;

  2. Template atau tema blog

  3. Mengubah PSD (Desain web) menjadi HTML

  4. dan sebagainya.

Hal dasar yang harus kita ketahui setelah memahami cara menulis HTML di CSS adalah selektor. Selektor adalah katakunci dan simbol yang digunakan pada CSS untuk menyeleksi atau memilih elemen HTML. Nilai warna dalam CSS dapat kita berikan dengan berbagai cara. Salah satunya dengan menuliskan nama warna dalam bahasa inggris.

Contoh :

  • Selektor elemen: <p>, <div>, <h1> sampai <h6>
  • Selektor kelas: .classname
  • Selektor ID: #idname
  • Selektor atribut: [attribute="value"]
  • Selektor pseudo-class: :first-child

Ada beberapa cara untuk menentukan warna dalam CSS:

    Nama warna dalam bahasa Inggris: red, blue, green
    Nilai heksadesimal: #FF0000 (merah)
    RGB: (255, 0, 0) (merah)
    RGBA: (255, 0, 0, 0.5) (merah dengan transparansi 50%)
    HSL: hsl(0, 100%, 50%) (merah)
    HSLA: hsla(0, 100%, 50%, 0.5) (merah dengan transparansi 50%)
